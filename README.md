Cozy Seasons (Pebble Watchface)

A seasonal Pebble watchface for Basalt (144x168) aka Pebble Time/Time Steel featuring dynamic backgrounds, lightweight particle effects, and an optional Low Vision Mode for improved readability.

✨ Features

Seasonal backgrounds (Spring, Summer, Autumn, Winter) with 24 hourly images each

Tap/shake particle effects

Spring: petals

Summer: seeds (day), fireflies (night)

Autumn: leaves

Winter: snow

12-hour & 24-hour support (handles edge cases like midnight correctly)

Low Vision Mode with high-contrast panels and improved layout

Bluetooth disconnect indicator with optional vibration

Battery bar with clear charge visualization

Customizable settings via Clay

📱 Platform

Pebble Basalt (color, 144x168)

Built with Pebble SDK + Clay

🚀 Installation

Available via the Pebble app store

Or sideload using the .pbw file from releases

🧠 Design Goals

Clean, glanceable time display

Consistent layout across modes

Lightweight performance (no continuous animation)

Seasonal aesthetic without sacrificing readability
